GIT笔记

2019-11-01 1 min read # git

git init

git config --global user.name "用户名" （这里的用户名是你要设置的git的全局姓名或昵称）

git config --global user.email "邮箱地址" （设置的全局邮箱地址）

git remote add origin https://git.oschina.net/username/cccc.git （你的远程仓库地址地址）

git pull origin master 同步

//输入账号密码

git status 查看是否有文件改变

git add . 如果有改动就执行这句

git commit -m "one" 添加备注

git push -u origin master 上传到码云

git reflog

git reset --hard HEAD ^ 回退上个版本 ^一个表示一个版本 git reset --hard HEAD ~ 100 回退上100个版本

git reset --hard 版本号 回退版本 输入版本号

git pull origin master --allow-unrelated-histories 强制pull git仓库代码

git checkout *** 切换分支

git merge **** 合并某个分支到当前分支
